# Task Manager

This is a simple Task Manager web application where users can create, view, update, and delete tasks.
The project is built using a basic frontend and a Node.js backend with MongoDB for storing tasks.

## Tech Stack

Frontend

* HTML
* CSS
* JavaScript

Backend

* Node.js
* Express.js
* MongoDB

## Features

* Add new tasks
* View all tasks
* Update task status
* Delete tasks
* Tasks are stored in MongoDB database

## Installation

1. Clone the repository

git clone https://github.com/AkashTyagiCSIT/task__manager.git


2. Go to backend folder

cd backend

3. Install dependencies

npm install

4. Start the server

node server.js

Server will run on:
http://localhost:5000

Live Demo: https://task-manager-ku67-ha58jdaex-attyagi2002-1021s-projects.vercel.app
